Crypto++ 5.4

Crypto++ 5.4 was released on December 22, 2006. The 5.4 release added Salsa and Whirlpool 3.0 classes.

Release Notes

The release notes for Crypto++ 5.4 follows.

  • added Salsa20
  • updated Whirlpool to version 3.0
  • ported to GCC 4.1, Sun C++ 5.8, and Borland C++Builder 2006
