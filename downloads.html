The downloads page provides all downloads of the Crypto++ library. The earliest download available is Crypto++ 2.3 from January 1998. Crypto++ 1.0 was released in June 1995, but the download is no longer available.

The hashes provided with a download were generated from the original release files from the Crypto++ website. The hashes may differ from hashes generatd on Github or Sourceforge. Some links to external sites may no longer be available.

A releated page is Release Versioning on the Crypto++ wiki. The versioning page includes Git commits and Subversion revisions and tags when avilable.
